Programmi

Software da scaricare e installare sul proprio computer per lavorare in modo professionale. Editor, browser, strumenti di versionamento.

Se non hai ancora questi quattro programmi, non sei un vero studnete del Marconi

Gratis

Visual Studio Code

L'editor di codice più usato al mondo. Leggero, veloce, con migliaia di estensioni scaricabili gratuitamente. Supporta qualsiasi linguaggio di programmazione. Funziona su Windows, Mac e Linux. Abbiamo usato questo per fare tutto il progetto.

Windows / Mac / Linux
code.visualstudio.com
Open Source

Git

Il sistema di controllo versione usato praticamente da tutti gli sviluppatori del mondo. Si installa sul computer e permette di gestire le versioni del codice tramite terminale. Indispensabile per usare GitHub. Senza Git non si può fare git commit, git push ecc.

Windows / Mac / Linux
git-scm.com
Gratis

Google Chrome

Il browser preferito dagli sviluppatori grazie agli strumenti per sviluppatori (DevTools) integrati. Con F12 si apre un pannello dove si può ispezionare l'HTML, vedere il CSS applicato, debuggare il JavaScript e simulare come appare il sito su uno smartphone.

Windows / Mac / Linux
google.com/chrome
Freemium

Figma Desktop

Versione desktop di Figma. Funziona esattamente come la versione web ma è più veloce e reattiva, soprattutto con file di design grandi. Utile se si lavora spesso con Figma e si vuole una finestra dedicata senza aprire il browser.

Windows / Mac
figma.com/downloads

Strumenti più avanzati

Contali da solo i programmi

Questi programmi si usano di più in ambito professionale. Dato che non siamo professionisti, Non ci servono, però dai almeno gli abbiamo messi che è tanta roba.

Freemium

Postman

Strumento per testare le API REST. Un'API è come un "portale" che un server mette a disposizione per ricevere e inviare dati. Con Postman si possono inviare richieste HTTP (GET, POST, PUT, DELETE) e vedere la risposta del server. Molto usato nel backend development.

Windows / Mac / Linux
postman.com
Freemium

Docker Desktop

Permette di creare ambienti di sviluppo isolati chiamati "container". Ogni container ha le sue dipendenze e funziona allo stesso modo su qualsiasi computer. È la soluzione al classico problema "sul mio computer funziona ma sul tuo no". Standard nel mondo professionale.

Windows / Mac / Linux
docker.com
Freemium

TablePlus

Client per la gestione di database con un'interfaccia grafica moderna. Supporta MySQL, PostgreSQL, SQLite e altri. Permette di vedere i dati del database in tabelle, fare query SQL e modificare i record direttamente senza riga di comando. Molto più comodo di lavorare da terminale.

Windows / Mac
tableplus.com
A pagamento

Adobe Photoshop

Il software professionale per il ritocco fotografico e la grafica raster. È a pagamento (abbonamento mensile) ma è lo standard dell'industria per il design visivo. Per la maggior parte dei progetti scolastici si può usare Figma o strumenti gratuiti, ma Photoshop è una cosa che molte aziende richiedono.

Windows / Mac
adobe.com/photoshop
