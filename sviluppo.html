Versionamento e collaborazione

2 risorse

Il versionamento del codice permette di tenere traccia di tutte le modifiche e lavorare in gruppo senza sovrascrivere il lavoro degli altri. È una cosa fondamentale nel mondo del lavoro reale.

🐙
Freemium

GitHub

La piattaforma più usata per salvare e condividere codice online. Si basa su Git (il sistema di versionamento). Quasi tutti i progetti open source del mondo sono su GitHub. Noi abbiamo caricato questo progetto qui.

github.com
🦊
Open Source

GitLab

Alternativa a GitHub, open source e con funzionalità di CI/CD integrate gratuitamente. Molto usato nelle aziende che preferiscono tenere il codice su server propri (self-hosted). Ottimo per i progetti scolastici.

gitlab.com

Hosting e deploy

3 risorse

Per pubblicare un sito online bisogna caricarlo su un server. Questi servizi permettono di farlo in modo semplice, spesso gratuitamente per progetti piccoli come i progetti scolastici.

📄
Gratis

GitHub Pages

Permette di pubblicare siti web statici (HTML/CSS/JS) direttamente da un repository GitHub, gratis. Perfetto per i progetti scolastici. Questo sito potrebbe essere pubblicato così senza spendere niente.

pages.github.com
▲
Freemium

Vercel

Piattaforma per il deploy di siti web e applicazioni. Si collega a GitHub e pubblica automaticamente ogni volta che si fa un push. La versione gratuita è più che sufficiente per la maggior parte dei progetti.

vercel.com
🟢
Freemium

Netlify

Alternativa a Vercel, molto usata per i siti statici. Permette anche di gestire form di contatto e funzioni serverless senza configurare un backend. Deploy automatico da GitHub o drag and drop dei file.

netlify.com

Strumenti CSS e testing

3 risorse

Utilità online per scrivere CSS migliore, verificare la compatibilità con i browser e testare il codice al volo.

✒️
Freemium

CodePen

Editor online dove si scrivono HTML, CSS e JavaScript e si vede il risultato in tempo reale. Perfetto per fare prove rapide senza aprire VS Code. C'è anche una community enorme con migliaia di esempi interattivi da cui imparare.

codepen.io
🔍
Gratis

Can I Use

Controlla se una proprietà CSS o una funzionalità JavaScript è supportata da tutti i browser (Chrome, Firefox, Safari, Edge...). Indispensabile per non fare errori di compatibilità. Ci vuole un attimo a controllare.

caniuse.com
🎩
Gratis

CSS-Tricks

Il sito di riferimento per tutto quello che riguarda il CSS. Articoli tecnici approfonditi, guide su Flexbox e CSS Grid, almanacco completo delle proprietà. Abbiamo trovato qui la spiegazione di auto-fill con minmax che usiamo nella griglia del sito.

css-tricks.com
